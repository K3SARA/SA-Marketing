## ⚠️ Important Things to Know

### 1. Create a Backup TODAY

The app stores everything locally on your device. **If something goes wrong, you need a backup to recover your data.**

**How to backup:**

1. Open the app
2. Click "Backup" button in top right
3. A file will download (save it somewhere safe)
4. Do this at least once per week!

### 2. Test with Sample Data First

Before using with real bills:

1. Try creating a test bill with test products
2. Practice collecting payments
3. Try printing a receipt
4. Make sure everything works for you

### 3. Multiple Devices Same Data?

⚠️ **Important:** Each device keeps its own copy of data. If you use the app on both your phone and tablet, the data won't sync between them.

**Solution:**

- Use the backup feature to move data between devices
- Export from Device A, Import on Device B

### 4. Forgotten Password?

Click "Forgot Password / Reset Login" to:

- Create a new username and password
- Your bill history will stay safe!

## ❓ Common Questions

### Q: What if my phone gets lost?

A: As long as you created backups, you can restore on a new phone. Import the backup file and you're back in business.

### Q: Can I use this on 2 devices?

A: Yes, but they won't automatically sync. You need to backup from one device and import on the other.

### Q: What if the internet goes down?

A: No problem! The app stores everything locally. It works perfectly offline. When internet comes back, you can backup and share.

### Q: How do I print a receipt?

A: Create a bill, then click "Print Receipt" in the bill preview. Your browser will show a print dialog.

### Q: Can staff use this app?

A: Yes! If all staff use the same username/password on the same device, they can share. For more security, each staff member can have their own device with their own login.

### Q: How do I generate reports?

A: Click the "Reports" tab at the bottom. You can see:

- Daily sales
- Weekly sales
- Collection history
- Cheque details
- Inventory

## 🎓 Pro Tips

**1. Use customer names:**

- The app remembers customer names
- Start typing a name to see past customers
- Helps you track who owes money

**2. Add products quickly:**

- During billing, start typing a product name
- If it doesn't exist, you can quickly add it
- No need to go to inventory page every time

**3. Split payments:**

- Accept cash AND card for same bill
- Add multiple payment methods
- System automatically calculates what's left

**4. Cheque payments:**

- Record cheque details (number, date, bank)
- App tracks cheque status (pending/cleared)
- Generate cheque reports later

**5. Expense tracking:**

- Record expenses (fuel, packaging, etc.)
- Filter by date range
- See total expenses vs revenue
